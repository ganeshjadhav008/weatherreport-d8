INTRODUCTION
------------
  A Material created to get a Weather report


REQUIREMENTS
------------

  No special requirements.


INSTALLATION
------------

  * follow the following steps for installation
    - go to /admin/modules 
    - find Material Task module using filter 
    - check checkbox and click on install


PERMISSION
-------------
  * go to admin/people/permissions
  * Find group Material
  * add permission role wise

ACCESS FORM
-------------
  * The menu will be enabled in the Tool block. Menu name: Weather report
  * URL: /weather
